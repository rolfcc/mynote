## 3. E4X

JavaScript = ECMAScript  
E4X = JavaScript for XML (ECMAScript For XML)  

- E4X 是对 JavaScript 的新扩展。  
- E4X 向 JavaScript 添加了对 XML 的直接支持。  
- E4X 是正式的 JavaScript 标准。

`var x = new XML()` XML是一个对象！！！  

浏览器对 E4X 的支持是有限的。
