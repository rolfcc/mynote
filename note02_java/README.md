## 安装

<a href="https://www.oracle.com/technetwork/java/javase/downloads/index.html" target="_blank">Java SE 下载</a>

Windows 在安装后需要配置环境变量，命令行输入 `Java -version` 显示版本信息则成功。

Linux 库里一般有，直接安装即可。

## Java 编译和运行

|       命令        |             作用              |
| :---------------: | :---------------------------: |
| javac 文件名.java |             编译              |
|    java 文件名    | 运行class文件（不用输 .class) |
